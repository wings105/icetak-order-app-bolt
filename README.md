# iCetak Order App - Bolt Starter

This is the clean GitHub/Bolt starter for moving the AppDeploy order app to Supabase.

## Supabase project

Project ref: `buivecgahhmrhlmfujgt`

Public URL:

```txt
https://buivecgahhmrhlmfujgt.supabase.co
```

## Setup

1. Copy `.env.example` to `.env.local`.
2. Run:

```bash
npm install
npm run dev
```

3. Open the app and click **Test Supabase tables**.

## Bolt import path

Recommended:

```txt
GitHub repo -> Bolt import from GitHub -> connect Supabase -> test -> build features
```

Local ZIP import is okay for first test, but GitHub import is better once the repo exists because it becomes the source of truth.

## Important

Do not commit private keys or service-role keys.
